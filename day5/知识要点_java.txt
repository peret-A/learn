1. while 循环
	形式: while(条件){
			循环体
			}
	意义：当条件成立时，执行循环体的代码，当不满足时，循环结束。
	注：要确定条件是否可以结束，否则将会陷入死循环中。

2. do-while 循环 先执行循环体再判断条件是否成立，如果成立继续执行，如果不成立结束

	形式：do{
			循环体
			}while（条件）;
	注：相较于while循环来说，执行次序不一样
	do-while循环体至少执行一次。
3. for 循环 逻辑：当满足条件时，执行循环体，再进行迭代
				  当不满足条件时，循环结束。
				  
	形式：for(初始值;条件;迭达){
			循环体
			}
