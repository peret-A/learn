1. sql的通用语法：
	1. sql语句可以单行或者多行书写，以分号结尾
	2. sql可以通过空格或者缩进提升代码的可读性
	3. sql教育不分大小写 关键词建议大写.
	4. 注释：
		单行：--注释内容 或者 #注释内容
		多行：/*注释内容*/
2. sql的分类：
		ddl  数据定义语言 作用：定义表...
		dml  数据操作语言 作用：对数据增加删除修改
		dql  数据查询语言 作用：对数据进行查询
		dcl  数据控制语言 作用：用创建数据用户，控制数据库的访问权限

3. ddl 
		1.查询
			#查询所有数据库
			show databases
			#查询当前数据库
			select database();
			#创建
			create [if not exists] database 数据库名;
			#删除
			drop [if exists] database 数据库名;
			#使用
			use 数据库名;
